Space Battle

Craiova

This is my first project ever. In this project, I created a game In Visual Studio in C++ and the SFML program language. Space Battle is a co-op game in which you control a plane that can move in any direction you want and destroy the enemies which are randomly spawned from the right side of the screen. The players have a static number of hit-points(10), but the enemies have a random amount(1, 2, or 3 hitpoints). If the player destroys several enemies, the level of the plane increases and if he runs out of lives then the other player wins.
